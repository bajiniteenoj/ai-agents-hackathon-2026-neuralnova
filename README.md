# NeuralNova – AI Study Assistant

NeuralNova is an AI-powered study assistant built for the AI Agents Hackathon 2026 by HackIndia.

## Features

* AI-powered question answering
* Smart note summarization
* Voice interaction
* Productivity and study planning

## Tech Stack

* Python
* Streamlit
* Gemini API / OpenAI API

## Goal

Our goal is to help students improve learning and productivity using conversational AI and intelligent automation.

AI Agents Hackathon 2026 – HackIndia
